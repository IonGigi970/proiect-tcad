# foi-de-catalog/

Aici sunt foile de catalog (datasheet) pentru componentele folosite în proiect.

## Recomandare organizare
Poți organiza în subfoldere, de exemplu:
- `IC/` (ex: 4001)
- `Conectori/`
- `Potențiometru/`
- `LED/`
- `Pasive/`

## Notă
Dacă o componentă are datasheet doar online, poți adăuga un fișier text cu:
- numele componentei
- link
- data accesării
