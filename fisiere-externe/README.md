# fisiere-externe/

Acest folder conține resurse externe sau auxiliare folosite în proiect.

## P1 – pachet ECAD (TME / SamacSys)
Potențiometrul P1 folosește un model ECAD disponibil pe pagina produsului TME în secțiunea “ECAD Model”.
Modelul a fost realizat via SamacSys și este accesibil public.

### Ce conține de obicei pachetul
- footprint (PCB)
- simbol (schematic / part)
- modele 3D (opțional)
- tutorial PDF (instrucțiuni de import pentru mai multe programe CAD)

### Integrare
Integrarea footprint + simbol în OrCAD a fost realizată urmând tutorialul PDF inclus în pachet.

> Recomandare: păstrează pachetul în forma originală (fără a redenumi fișierele interne), ca să fie ușor de urmărit instrucțiunile.

## J1 – footprint custom (pitch 5 mm)
Proiectul include și un footprint/capsulă creat(ă) manual pentru conectorul J1 (pas pini 5 mm). XY301A_2pin.dra este fișierul care trebuie pus în folder-ul de footprints al OrCAD-ului.
